## 📁 文件列表

### 1. 条件独立性检验（带容差）  
**文件**: `Conditional Independence Test with Tolerance.pdf`  

- **研究问题**：在因果推断中，如何判断变量 `X` 与 `Y` 在给定混杂因子 `Z` 后是否独立？当真实关系很弱（条件互信息不严格为零）时，传统检验会错误地发现“显著”关系。  
- **方法**：提出一种带有**容差**的假设检验框架 —— 只有当条件互信息超过阈值 τ 时才认为有关联。利用随机化检验生成重采样数据，基于 `χ²` 统计量构造检验，并通过优化问题控制原假设下的 KL 散度上界。  
- **主要结果**：定理证明了所提方法的渐近有效性和尺寸可控性；仿真实验表明，当真实依赖强度低于容差时，检验能够正确控制第一类错误，高于容差时具有良好功效。  
- **关键词**：条件独立性检验、随机化检验、容差、χ² 统计量、互信息。

### 2. 半监督推断（Prediction‑Powered Inference）  
**文件**: `Prediction Powered Inference.pdf`  

- **研究问题**：在有大量无标签数据但仅有少量有标签数据时，如何利用机器学习模型辅助统计推断（如估计均值、构建置信区间）且不引入偏差？  
- **方法**：复现 [Angelopoulos et al., 2023] 的 Prediction‑Powered Inference 框架：用模型预测无标签数据得到“伪标签”，同时用有标签数据估计预测偏差并修正最终估计量，保证无偏性。  
- **主要结果**：推导了估计量的渐近方差，给出置信区间公式。仿真实验显示，当预测模型准确（如 `R²` 高）时，该方法能显著缩短置信区间宽度，提高检验功效；当模型不准时，性能不劣于传统方法。  
- **关键词**：半监督学习、预测驱动的推断、无偏估计、置信区间、机器学习辅助统计推断。

### 3. 自定义决策树与随机森林（Python 实现）  
**文件**: `costomized_decision_tree.py`  

- **功能**：从零实现 CART 决策树算法及随机森林的基础组件。  
- **实现细节**：  
  - 支持均方误差（MSE）分裂准则；  
  - 递归构建二叉树，包含树的生长、节点分裂、剪枝（通过 `max_depth` 限制）；  
  - 为随机森林预留了特征随机子采样接口（`max_features='sqrt'`）；  
  - 代码注释清晰，模块化设计，易于扩展（例如自定义损失函数、缺失值处理等）。  
- **使用价值**：深入理解树模型底层机制，可用于教学演示或作为自定义模型的范本。
